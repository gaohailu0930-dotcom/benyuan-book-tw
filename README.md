# 本源幣：重寫經濟的地基（繁體精校版）

> 一部獨立完成的經濟理論著作，全書免費閱讀與傳播。

幾百年來，有一條假設被所有人繼承，卻從沒有人認真質疑過：**貨幣，必須為正。**

這本書從這一個念頭出發，推演出一套允許帳戶為負、而全社會總帳恆為零的記帳體系——**本源幣**。它不止步於一個口號：從生產、勞動、環境、公共服務、人口，到創新、信用、日常生活、治理與文明定位，全書十四章，幾乎不用一個公式，只用邏輯和日常的語言，講給每一個普通人聽。

— 高海璐 著
